# Collector's Search

Инструмент для поиска коллекционных предметов и сетов в инвентаре Dota 2 через веб-интерфейс.

## Основные возможности

- **Умный поиск сетов**: Использует базу данных (`dota_sets_db.json`) для поиска всех частей сета, даже если их названия не содержат имя самого сета (актуально для Collector's Cache).
- **Поиск Аркан и Личностей**: Отдельные базы `arcanas_db.json` и `personas_db.json` для точного определения наличия редких предметов в инвентаре.
- **Интеграция с Торговой площадкой Steam**: Все предметы, которые можно обменять (Tradable) и продать (Marketable), собираются в отдельную категорию «Предметы» с актуальными ценами из API `market.dota2.net`.
- **Поддержка переименованных предметов**: Автоматически извлекает оригинальное название предмета из метаданных Steam (`fraudwarnings`), если он был переименован ярлыком.
- **Очистка префиксов качества**: Игнорирует префиксы типа `Inscribed`, `Autographed`, `Genuine` и др. при сопоставлении с базой сетов.
- **Парсинг цен для сетов**: Подтягивает ориентировочные цены полных сетов с `collectorsshop.ru`.
- **Поддержка больших инвентарей**: Реализована автоматическая пагинация для обработки профилей с тысячами предметов.
- **Генерация отчета**: Удобный вывод результатов с возможностью копирования краткой сводки (общее количество сетов, цены, арканы) для передачи на торговые платформы.

## Структура проекта

- `web_app.py` — Основное приложение с красивым веб-интерфейсом, статистикой и фильтрами по категориям.
- `inventory_finder.py` — Ядро проекта, логика матчинга предметов и парсинг инвентаря Steam.
- `dota_sets_db.json` — База данных составов сетов.
- `arcanas_db.json` — База данных Аркан.
- `personas_db.json` — База данных Личностей.

## Установка и запуск

### Требования
- Python 3.10+
- Публичный инвентарь в Steam (профиль должен быть открыт).

### Запуск веб-интерфейса
1. Запустите сервер:
   ```bash
   python web_app.py
   ```
2. Откройте `http://127.0.0.1:8000` в браузере.
3. Вставьте ссылку на Steam инвентарь и нажмите "Искать".

## Ограничения
- Steam API может временно заблокировать запросы (ошибка 429) при слишком частом парсинге огромных инвентарей.
- Если инвентарь скрыт настройками приватности Steam, поиск будет невозможен.
