# Collector's Search

Инструмент для поиска коллекционных предметов и сетов в инвентаре Dota 2. Поддерживает поиск как через CLI, так и через веб-интерфейс.

## Основные возможности

- **Умный поиск сетов**: Использует базу данных (`dota_sets_db.json`) для поиска всех частей сета, даже если их названия не содержат имя самого сета (актуально для Collector's Cache).
- **Поддержка переименованных предметов**: Автоматически извлекает оригинальное название предмета из метаданных Steam (`fraudwarnings`), если он был переименован ярлыком.
- **Очистка префиксов качества**: Игнорирует префиксы типа `Inscribed`, `Autographed`, `Genuine` и др. при сопоставлении с базой сетов.
- **Парсинг цен**: Подтягивает ориентировочные цены с `collectorsshop.ru`.
- **Поддержка больших инвентарей**: Реализована пагинация для обработки профилей с 2000+ предметами.

## Структура проекта

- `inventory_finder.py` — Ядро проекта и логика матчинга.
- `web_app.py` — Веб-интерфейс (Flask) для удобного поиска.
- `build_set_db.py` — Скрипт для автоматического наполнения базы данных сетов с Liquipedia.
- `audit_db.py` — Инструмент для проверки целостности и чистоты базы данных.
- `dota_sets_db.json` — База данных составов сетов.
- `names_example.txt` — Список названий сетов для поиска (по умолчанию).

## Установка и запуск

### Требования
- Python 3.10+
- Публичный инвентарь в Steam.

### Веб-версия
1. Запустите сервер:
   ```bash
   python web_app.py
   ```
2. Откройте `http://127.0.0.1:8000`.
3. Вставьте ссылку на инвентарь и нажмите "Искать".

## База данных сетов
Если база данных пуста или вы хотите её обновить:
1. Запустите `python build_set_db.py`. Скрипт соберет данные о сетах и их частях.
2. Проверьте результат с помощью `python audit_db.py`.

## Ограничения
- Steam может временно заблокировать запросы (ошибка 429) при слишком частом использовании.
- Если инвентарь скрыт настройками приватности, поиск невозможен.
